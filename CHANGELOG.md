# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2024-12-11

### Fixed
- Respects section's "Default Entry Placement" setting (Beginning/End) when auto-parenting entries

## [1.0.0] - 2024-12-08

### Added
- Initial release
- Automatic parenting of new entries based on Entry Type configuration
- Support for unlimited nesting depth (multi-level hierarchies)
- Three parent selection strategies: Most Recently Updated, First Created, Specific Entry
- Visual settings interface with drag-and-drop ordering
- Hierarchy preview in settings
- Project Config support for environment sync
- Craft CMS 5.0+ compatibility
